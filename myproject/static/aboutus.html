Осцилло́граф (лат. oscillo — качаюсь + греч. γραφω — пишу) — прибор, предназначенный для исследования электрических сигналов во временно́й области путём визуального наблюдения графика сигнала на экране либо записанного на фотоленте, а также для измерения амплитудных и временны́х параметров сигнала по форме графика. Современные осциллографы позволяют разворачивать сигнал гигагерцовых частот. Для разворачивания более высокочастотных сигналов можно использовать электронно-оптические камеры.
